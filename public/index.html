Please answer and complete the following application form. Take in account the following guidelines:

1. All fields are required except the ones marked as (optional).

2. If the applicant does not host an online portfolio website, he/she must provide a list of URLs using the "Other Links (optional)" field in this form. We recommend to use vimeo for video samples, flickr and soundcloud for audio samples.

3. If the applicant does have an online portfolio and still wants to provide extra URLs for additional work samples he/she is free to do so by using the "Other Links (optional)" field in this form.

4. The Project Proposal PDF should be uploaded as a clear document containing at least the following items:

  • Genral Description of the Project
  • Conceptual Framework
  • Technical Description (software / hardware)
  • Based on CAC_LAB's current research interests (as expressed in the open call document under the section "About CAC_LAB"), explain how your project contributes to the development of the LAB's present needs (Technically and Conceptually).
  • Detailed Time Schedule and Development Plan
  • Budget

5. Finally, please upload your CV / Resume as a PDF.
